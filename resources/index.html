Excel文法词汇索引 4.0

下载地址：

自制Excel文法词汇索引4.0 现在已经支持 Windows 和 Mac 使用。

  • 适用于 Windows （GBK 编码）： 点击下载(Win)

  • 适用于 Mac (UTF-8 编码)：点击下载(Mac)

使用教学：

  • Windows ：

    下载后将档案解压缩到你想要保存的位置，打开「文法词汇索引4.0 - Windows.xlsx」，修改文档保存的位置就可以使用查找功能查询使用了。

  • Mac ：

    下载后请将文件解压在 “图片”(Pictures) 目录，确保「にほんご」文件夹在“图片”的根目录。

    打开「文法词汇索引4.0 - Mac.xlsx」然后修改文档保存位置，将你Mac电脑的“用户名”替换到文档中如下图的位置后，就可以查询使用了。例如：file:///Users/mymac/Pictures/にほんご
