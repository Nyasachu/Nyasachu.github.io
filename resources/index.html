Excel文法词汇索引 4.0

下载地址：

自制Excel文法词汇索引4.0 现在已经支持 Windows 和 Mac 使用。

  • 适用于 Windows / Mac 共用： 点击下载

使用教学：

  • Windows ：

    下载后将档案解压缩到你想要保存的位置，打开「文法词汇索引4.0 - Windows.xlsx」，修改文档保存的位置就可以使用查找功能查询使用了。

  • Mac ：

    • 方法1：放置于”图片“目录：下载后请将文件解压在 “图片”(Pictures) 目录，确保「にほんご」文件夹在“图片”的根目录。打开「文法词汇索引4.0 - Mac.xlsx」然后修改文档保存位置，将你Mac电脑的“用户名”替换到文档中如下图的位置后，就可以查询使用了。例如：file:///Users/mymac/Pictures/にほんご

    • 方法2：自定义存放位置：将“にほんご”文件夹存放在任意位置，并准确填入文件夹路径。填写完成后请点击一次你输入的路径，并在提示框中点击“允许”。如果能够在Finder中正常打开对应的文件夹，则代表路径设置正确。但是，可能每次重新打开Excel后，都需要点击一次这个路径并给与访问授权。
